## Setup Instructions

1. Upload all files to your web hosting
2. Replace placeholder images in `/images/` with actual images:
   - `csgoroll-logo.svg` → Your actual CSGORoll logo
   - `csgoroll-hero.svg` → Hero section image
   - `favicon.svg` → Site favicon (also generate PNG versions)
3. Configure your web server to serve `index.html` for the root URL
4. Submit `sitemap.xml` to Google Search Console

## Image Replacement

The current images are SVG placeholders. Replace them with:

- **Logo**: PNG/SVG format, recommended size 150x40px
- **Hero Image**: PNG/JPG format, recommended size 500x400px
- **Favicon**: Generate favicon files using a tool like [RealFaviconGenerator](https://realfavicongenerator.net/)
